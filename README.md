# 0G Neumorphic UI Toolkit 🪐

A plug-and-play, zero-download UI toolkit that instantly transforms your Web3 dApp into a sleek, **0G-branded Neumorphic** experience. 

Built for the [Zero Gravity (0G)](https://0g.ai/) ecosystem, this toolkit injects the official 0G brand colors, typography, and a dynamic light/dark Neumorphic design system directly into your frontend with a single line of code.

## ✨ Features

* **Zero Configuration:** No `npm install`, no complex setups. Just paste the CDN link and go.
* **True Neumorphism:** Automatically applies mathematically accurate extruded and pressed shadows for both Light and Dark modes.
* **0G Brand Compliant:** * Primary accent color: Deep Purple (`#9200E1` to `#E3C1FF` ramp).
  * Typography: Automatically imports **Regola Pro** for UI and **Geist Mono** for code.
* **Auto Theme Detection:** Detects the user's system preferences (`prefers-color-scheme`) and switches between light and dark modes seamlessly.
* **Smart Logo Injection:** Automatically swaps your placeholder images with the official 0G Light or Dark logos based on the active theme.

---

## 🚀 Quick Start (Usage)

To apply the 0G Neumorphic style to your project, simply add this single `<script>` tag inside the `<head>` of your HTML file:

```html
<script src="[https://cdn.jsdelivr.net/gh/YRedRavenLabs/0g-ui-toolkit@main/styler.js](https://cdn.jsdelivr.net/gh/RedRavenLabs/0g-ui-toolkit@main/styler.js)"></script>
